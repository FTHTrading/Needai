## 🎯 Deployment Options

### **Option A: GitHub Integration (Recommended)**
Automatic deployments on every push.

1. **Connect GitHub repo to Cloudflare Pages:**
   - Go to: https://dash.cloudflare.com/pages
   - Click "Create a project" → "Connect to Git"
   - Select repository: `unykornai/assistant-vanity-`
   - Branch: `main`

2. **Build settings:**
   ```
   Framework preset: Next.js
   Build command: npm run build
   Build output directory: .next
   Root directory: (leave blank)
   ```

3. **Environment variables:**
   Add any `.env` variables:
   ```
   TELNYX_API_KEY=your_key_here
   NEXT_PUBLIC_API_URL=https://vanity.ai
   ```

4. **Deploy:** Click "Save and Deploy"

---

### **Option B: Direct Deploy via Wrangler CLI**
Manual deployment from local machine.

1. **Install Wrangler:**
   ```powershell
   npm install -g wrangler
   ```

2. **Login to Cloudflare:**
   ```powershell
   wrangler login
   ```

3. **Build for production:**
   ```powershell
   npm run build
   ```

4. **Deploy:**
   ```powershell
   npx wrangler pages deploy .next --project-name=vanity-ai
   ```

## 🔧 Next.js on Cloudflare Pages Configuration

### **Add to `next.config.ts`:**
```typescript
const nextConfig = {
  output: 'export', // Static export for Cloudflare Pages
  trailingSlash: true,
  images: {
    unoptimized: true, // Cloudflare doesn't support Next.js Image Optimization
  },
};

export default nextConfig;
```
