# ⚠️ VANITY NUMBER ACCURACY & COMPLIANCE GUIDE

**Critical: Billboard claims vs. actual dial numbers**

---

## 🎯 THE CORE RULE

**What's on the billboard ≠ What they dial**

This is **100% legal and standard practice**, but you must structure it correctly.

---

## ✅ CORRECT STRUCTURE (DO THIS)

### **Headline = Message (can say anything true)**
### **Call instruction = Must be accurate digits**

**Example:**
```
NEED MONEY?
CALL 888-CASH-AI
```

- ✅ Headline asks a question (true statement)
- ✅ Dial instruction is accurate (888-678-0645)
- ✅ No claim that "NEED-MONEY" dials anywhere

---

## ❌ INCORRECT STRUCTURE (DON'T DO THIS)

```
CALL NEED-MONEY
```

This claims that "NEED-MONEY" is dialable.
Unless the digits literally spell that, this is **false advertising**.

---

## 🧠 THE POWER OF TRUE "NEED" NUMBERS

### **Why 844-NEED and 912-NEED are special:**

These are **universal intake numbers**. Billboard says:
```
NEED HELP?
CALL 844-NEED
```

**What happens:**
1. Caller dials `844-669-6333`
2. AI answers: "What do you need help with?"
3. Caller says: "Money" / "Lawyer" / "Storm damage" / "HVAC"
4. AI routes to appropriate vertical

**This is MORE POWERFUL than vertical-specific numbers** because it captures ALL intent.

## 🛡️ COMPLIANCE CHECKLIST

Before printing ANY billboard:

- [ ] Does the **dial instruction** match actual digits?
- [ ] If using "NEED [SOMETHING]", is it a headline/question OR a true NEED number?
- [ ] Is the vanity word actually dialable on the number shown?
- [ ] Would an FTC/FCC auditor approve this?

## 📞 SAFE PATTERNS (ALWAYS WORK)

### **Pattern 1: Question Headline**
```
[QUESTION ABOUT NEED]
CALL [ACCURATE NUMBER]
```

### **Pattern 2: Statement + Accurate Number**
```
[TRUE STATEMENT]
CALL [ACCURATE NUMBER]
```

### **Pattern 3: Pure Vanity (Word = Digits)**
```
[WORD THAT MATCHES NUMBER]
CALL [SAME WORD WITH AREA CODE]
```

---

## ⚠️ DANGEROUS PATTERNS (DON'T USE)

### **❌ Pattern 1: False Dial Claim**
```
CALL NEED-MONEY
```
Unless the number ends in 6333-66639, this is **false**.

### **❌ Pattern 2: Implied Dialability**
```
NEED MONEY
888-555-1234
```
This visually implies "NEED MONEY" connects to that number.
Only safe if using a true NEED number (844-NEED, 912-NEED).

## 📝 APPROVAL PROCESS

**Before any billboard goes live:**

1. **Check digits:** Does the vanity word match the number?
2. **Check claim:** Are we saying it dials what it actually dials?
3. **Check structure:** Headline vs. dial instruction separated?
4. **Legal review:** Would this pass FTC/FCC scrutiny?

**If YES to all 4:** ✅ **APPROVED**
**If NO to any:** ❌ **REVISE**

## 🚨 CORRECTION PROTOCOL

3. Restructure using compliant pattern
